# Bundle G — Historical Intelligence Layer

Bundle G uses shared trace, coverage, limitation, finding, and validation contracts. Comparative, cross-era, pattern, civilization, institutional, strategic, trend, counterfactual, and theory analyses are deterministic rule-bound result layers. Synthesis orders findings stably; validation rejects missing traces. No LLM, NLP, embeddings, semantic similarity, probability, external data, or speculative alternate history is used.
