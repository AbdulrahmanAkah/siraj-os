# Historical Interpretation

## Boundary

Historical interpretation is deterministic composition. It repeats existing
reasoning candidate statements; it does not generate interpretive prose.

One record is created per narrative record. It preserves reasoning chain
IDs, narrative record IDs, causal relations whose exact cause or effect text
occurs in source statements, temporal relations touching source events,
resolved evidence IDs, and original evidence reference IDs.

Records preserve narrative position. IDs derive from the full trace material.

Validation requires existing statements, reasoning and narrative links,
resolved evidence, original source references, unique IDs, stable ordering,
and count consistency. No semantic interpretation, NLP, LLM, probability,
or external access is permitted.
