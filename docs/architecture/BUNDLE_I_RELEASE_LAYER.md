# Bundle I — SIRAJ 1.0 Release Layer

Bundle I adds typed release configuration, policy enforcement, bootstrap, health, `/v1` public API contracts, CLI v2, static plugins, dry-run migrations, neutral deployment manifests, deterministic release packaging, operator documentation, and governance gates. The selected code version is `0.1.0-rc.1`; it is a release candidate, not a 1.0.0 declaration.
