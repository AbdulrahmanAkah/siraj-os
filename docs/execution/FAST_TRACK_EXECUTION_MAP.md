# SIRAJ Fast-Track Execution Map

Status: ACTIVE
Owner: SIRAJ program
Canonical program record: PROJECT_PROGRESS.md

## North Star

Build an automated, evidence-traceable video production line covering Islamic
history from the creation of Adam through the Day of Judgment.

The system must transform Islamic sources into:

Source package
-> claims and evidence
-> knowledge structure
-> documentary plan
-> Arabic script
-> scenes and storyboard
-> visual assets
-> voice
-> subtitles
-> rendered video
-> verification
-> publication package

## Short-term objective

Produce the first reviewed and publishable documentary:

Episode 001 - The Creation of Adam: What the Islamic Sources Establish

The short-term objective is not Gold-20. Gold-20 is a bounded extraction-quality
and regression gate inside the knowledge workstream.

## Long-term objective

Create a repeatable production factory for a chronological Islamic-history
series and reuse the verified knowledge for documentaries, shorts, articles,
podcasts, courses, and search.

## Execution strategy

1. Build one end-to-end vertical slice before broadly perfecting every layer.
2. Extend the canonical pipeline; never create a duplicate production pipeline.
3. Use quality gates instead of requiring theoretical completion of each phase.
4. Permit reviewed source packages as a canonical human-review boundary.
5. Record every claim, source, evidence span, confidence, decision, and artifact.
6. Keep Gold-20 within a fixed work-in-progress limit.

## Work allocation

- 60 percent: first documentary vertical slice.
- 25 percent: real media execution.
- 15 percent: extraction quality and Gold-20.

## Delivery gates

### Gate 0 - Program reset

- Execution map stored.
- Episode 001 manifest stored.
- Pipeline readiness audited.
- Real media gaps listed.

### Gate 1 - Evidence ready

- Reviewed source package.
- Evidence-linked claims.
- Conflict and uncertainty labels.
- Human source approval.

### Gate 2 - Script ready

- Documentary plan.
- Arabic narration script.
- Complete claim-to-paragraph traceability.
- Script approval.

### Gate 3 - Visual ready

- Scene for every narration unit.
- Storyboard and visual strategy.
- Visual religious-safety approval.
- Complete asset manifest.

### Gate 4 - Render ready

- Voice generated.
- Visual assets resolved.
- Subtitles generated.
- FFmpeg render manifest valid.

### Gate 5 - Publishable

- Final MP4 generated.
- Documentary verification passes.
- Credits and citations included.
- Human final approval.
- Reproduction manifest stored.

## Immediate priority

Episode manifest
-> reviewed source pack
-> script package
-> media adapters
-> FFmpeg render
-> final MP4

## Definition of done

The fast-track phase is complete only when SIRAJ produces one reviewed,
publishable MP4 from a stored source package and can reproduce it from a
versioned manifest.
