# Siraj Engine Specification

Version: 1.0
Status: Approved

---

# Mission

Siraj is a Knowledge Factory.

Its purpose is to transform trusted sources into reusable Knowledge Assets,
then automatically render those assets into different forms of content.

Knowledge is produced once.

Content is rendered many times.

---

# Core Principle

The Knowledge Asset is the primary production unit.

Everything else consumes or produces Knowledge Assets.

# Human Role

Human is Editor-in-Chief.

The system performs all production work.

Human intervention should decrease over time.

Ultimate goal:

Topic

↓

Press Start

↓

Published Content

---

# Design Rule

Every component must answer one question:

"Does this make autonomous knowledge production better?"

If not,
it does not belong in Siraj Engine.
